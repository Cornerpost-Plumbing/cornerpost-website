# Cornerpost Plumbing Website

Version 1.0 website for Cornerpost Plumbing.

## Project structure

- `index.html` - homepage
- `services.html` - services page
- `about.html` - company philosophy and owner accountability
- `contact.html` - service request form
- `reviews.html` - customer reviews page
- `css/` - site styles
- `js/` - configuration, navigation, services, reviews, and site behavior
- `apps-script/` - Google Apps Script used by the service request workflow
- `favicon/` - favicon and app icon files
- `images/branding/` - website-ready logo assets
- `images/hero/` - homepage and landscape hero images
- `images/about/` - About page photography
- `images/services/` - service-specific photography
- `images/work/` - completed-work photography
- `images/vehicles/` - truck and vehicle photography
- `images/placeholders/` - temporary placeholder artwork

Business-wide settings are centralized in `js/config.js`. Service content and service image paths are maintained in `js/services.js`.
